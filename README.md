# FlowTask
FlowTask là một app desktop dành cho những người muốn quản lý các dự án phân chia công việc và theo dõi tiến độ hợp lý, rõ ràng. App sẽ cho phép người dùng phân chia công việc, theo dõi tiến độ đảm bảo đúng deadline, nhắc nhở/ thông báo những khoảng thời gian trưởng nhóm setup khi các thành viên chưa hoàn thành, nếu có nội dung nào vừa được nộp app sẽ hiển thị dự án nào vừa có cập nhật mới ở phần nào, hiển thị trạng thái công việc trực quan.

### Tech Stack
- Ngôn ngữ: Java
- Framework: JavaFX 
- Database: SQLite
- Thư viện cần dùng:
+ Spring Boot
+ Spring Data JPA
+ Hibernate
+ Spring Security + JWT
+ Bootstrap
+ Axios
+ SortableJS
+ FullCalendar
